Record some common shotcut keys of PR

<!-- more -->

| shotcut keys          | brief                      |
| :-------------------- | -------------------------- |
| c                     | 剪裁工具(cut)              |
| v                     | 选择工具                   |
| o/i                   | 选择出点(out)/选择入点(in) |
| Alt+mouse             | 拖动复制剪辑               |
| left/right            | 向左/右前进一帧            |
| <--j/k/l-->           | 倒放/暂停/播放             |
| +/-                   | 伸缩时间条                 |
| M/ctrl+M/ctrl+shift+M | 添加标签/跳转标签          |
|                       |                            |
